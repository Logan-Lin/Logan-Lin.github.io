Profile

Projects

Prompt Genius: Prompt Library for AI ChatBots
Feb'23 - Present

All-purpose prompts with hierarchical classification system and multi-language support. Help users discover functions of NLP ChatBots like ChatGPT.
Build the web application with Flask and HTML. Also build a corresponding management system with Electron.

Spatial-temporal Traffic Big Data Mining Platform
Jan'21 - Present

Designed an integrated platform for lubricating the process of spatial-temporal traffic big data mining, including data pre-processing, storing and prediction.
Implemented the platform based on Kafka, Spark and HBase to achieve real-time processing on large-scale spatial-temporal traffic data. Paired with an interactive GUI for intuitive data analysis.

Advisor: Prof. Huaiyu Wan, Prof. Youfang Lin
Dynamic Prediction Algorithm of Airline Catering Backup Quantity
Dec'20 - Present

Worked on predicting the airline catering backup quantity to help airline companies reduce economic and productivity losses.
Guide teammates to design a hierarchical RNN for modeling the correlation between prediction targets and long-range historical records.

Advisor: Prof. Huaiyu Wan
Highway Toll Rate Optimization System Based on Traffic Big Data
Jan'20 - Dec'20

Designed a traffic flow simulation system based on historical traffic trajectories and optimal cost algorithm. Aims to help urban managers balance highway toll station income and traffic flow.
Suggestion for the toll rate of Nansha Bridge accepted by Road Network Center of the Ministry of Transport, China. One related patent applied and is under audit.

Advisor: Prof. Huaiyu Wan, Prof. Youfang Lin
